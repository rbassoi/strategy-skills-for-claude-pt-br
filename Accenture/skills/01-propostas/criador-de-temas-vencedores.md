# Criador de Temas Vencedores

## Quando Usar

Use esta skill quando a equipe de pursuit precisa de posicionamento diferenciado antes de escrever a proposta. Adequada para qualquer negócio onde o cliente tem múltiplos fornecedores para escolher e mensagens genéricas de "capacidade ponta a ponta" não vencerão.

## Abordagem Accenture

Construa os temas a partir dos problemas do cliente para fora, não das capacidades da Accenture para dentro. Cada tema deve ser específico, mensurável e competitivamente diferenciado. Os temas devem aparecer de forma consistente em todas as seções da proposta.

## Fluxo de Trabalho

1. Identifique os 3 principais problemas de negócio do cliente no RFP ou briefing.
2. Mapeie cada problema para uma força específica ou capacidade única da Accenture.
3. Crie temas usando a estrutura: [Problema do cliente] → [Diferencial da Accenture] → [Resultado para o cliente].
4. Valide cada tema contra três critérios: específico (sem afirmações genéricas), mensurável (inclua uma métrica), diferenciado (não pode ser dito por todo concorrente).
5. Identifique provas ou referências de credenciais para cada tema.
6. Elabore linguagem de ghost: posicione os pontos fortes da Accenture sem nomear o concorrente.
7. Sinalize vitórias recentes de concorrentes no setor do cliente que precisam ser abordadas.

## Formato de Saída

```markdown
# Temas Vencedores

## Contexto do Cliente
[3 principais problemas de negócio do cliente extraídos do RFP ou briefing.]

## Temas Vencedores
| Tema | Problema do Cliente | Diferencial da Accenture | Resultado para o Cliente | Prova |
|---|---|---|---|---|

## Resumos em Uma Linha
[Uma frase por tema para uso no sumário executivo e carta de apresentação.]

## Sinais de Ghost
[Linguagem que destaca os pontos fortes da Accenture de forma a excluir sutilmente os concorrentes — sem nomeá-los.]

## Ameaças Competitivas a Abordar
[Vitórias recentes de concorrentes neste setor do cliente que a proposta deve contra-argumentar.]
```

## Padrão de Qualidade

- Nenhum tema pode usar: "ponta a ponta", "holístico", "alavancando sinergias" ou qualquer afirmação que todo concorrente poderia fazer.
- Cada tema precisa de uma métrica ou resultado, não apenas uma descrição de capacidade.
- Os temas devem ser rastreáveis por toda a proposta — não apenas no sumário executivo.
- Inclua IP específico de setor, alianças de ecossistema ou entrega local com expertise global onde relevante.
